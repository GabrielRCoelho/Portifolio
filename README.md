## Projeto - Portifólio

Este projeto consiste no desenvolvimento de um portfólio pessoal com o objetivo de apresentar meus projetos, habilidades e informações de forma clara e organizada.

A aplicação foi construída utilizando HTML e CSS, com foco na criação de um layout moderno, responsivo e visualmente agradável, proporcionando uma boa experiência para o usuário.

Além disso, o projeto busca aplicar boas práticas de estruturação de código e organização de componentes, servindo também como forma de consolidar conhecimentos em desenvolvimento front-end.
