## Projeto - Portifólio

Este projeto consiste no desenvolvimento de um portfólio pessoal com o objetivo de apresentar meus projetos, habilidades e informações de forma prática.

A aplicação foi construída utilizando HTML e CSS, com foco na criação de um layout moderno e visualmente agradável.

Além disso, o projeto busca aplicar boas práticas de estruturação de código e organização dos elementos, servindo também como forma de consolidar conhecimentos em desenvolvimento front-end.
