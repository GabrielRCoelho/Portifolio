# Portfólio

Este projeto consiste no desenvolvimento de um portfólio pessoal com o objetivo de apresentar meus projetos, habilidades e informações de forma clara e organizada.

A aplicação foi construída utilizando HTML, CSS e JavaScript, com foco na criação de um layout moderno e visualmente agradável, explorando conceitos de estruturação de páginas web, estilização e interatividade básica.

O projeto também serviu como prática para organização de código no front-end, separação de arquivos e construção de uma interface simples para navegação entre seções.

## Funcionalidades
- Seção de apresentação (sobre mim)
- Exibição de projetos
- Navegação por âncoras
- Layout organizado por seções
## Tecnologias utilizadas
- HTML
- CSS
## Melhorias futuras
- Melhorias no layout e design
- Adição de animações e interações
